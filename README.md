## üõ†Ô∏è Installation

- **Clone the Repository (use HTTPS if preferred)**

  ```bash
  git clone git@github.com:shricodev/portfolio.git
  ```

- **Navigate to the Directory**

  ```bash
  cd portfolio
  ```

- **Install Dependencies**

  ```bash
  npm install
  ```

- **Start the Development Server**

  ```bash
  npm run dev
  ```
